# Blog acessível sobre o São Paulo Futebol Clube
# Sobre:
Refatoração de um site do **Alura Start**, baseando-se e inspirando-se em elementos adicionados pelo professor durante as aulas.
Nele foi usado recursos de **HTML**, **CSS**, **JS** e das bibliotecas **Bootstrap** e **ScrolReveal** via **cdnjs**.
## Recursos acessíveis:
Disponibilizamos a possibilidade de aumentar a fonte e o contraste das cores, justamente para aumentar a visibilidade e legibilidade para todos os usuários.
As ferramentas foram:
- Atributos Aria;
- Alt;
- Tab-index;
- Menu de acessibilidades.
